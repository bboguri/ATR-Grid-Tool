## 🛠 ATR 动态网格实战 SOP

### 1. 标的诊断 (判定“心跳”)
使用计算器得出 **ATRP (ATR占比)**，根据其“性格”匹配参数：
- **公式**: $$ATRP = \frac{ATR(14)}{Price} \times 100\%$$

| ATRP 数值 | 波动等级 | 风险倍数 (M) | 确认比例 (R) | 核心建议 |
| :--- | :--- | :--- | :--- | :--- |
| **< 1.5%** | 低波动 | 2.0 | 0.5% | 空间小，适合宽基 ETF 大波段 |
| **1.5% - 3.0%** | 中波动 | 1.5 | 0.8% | 稳健型，适合大多数蓝筹/ETF |
| **3.0% - 5.0%** | 高波动 | 1.2 | 1.0% | **黄金区间**，套利效率最高 |
| **> 5.0%** | 极端波动 | 1.0 | 1.5% | 避开财报，拉开间距防御 |

### 2. 参数填报逻辑
- **买入步长**: $-(ATRP \times M)\%$ 
- **买入确认 (反弹)**: $+R\%$
- **卖出步长**: $+(ATRP \times M + 摩擦冗余)\%$ 
  - *注：个股冗余建议 0.5%，ETF 建议 0.2%。*
- **卖出确认 (回落)**: $-R\%$

## ⚠️ 风险管控 (生命线)
1. **破位休眠**: 股价有效跌破 **MA60** 或价格下限，立即停止网格。
2. **财报避险**: 财报发布前 3 个交易日手动关闭，防止单边跳空。
3. **现金流管理**: 确保可用现金能覆盖至少 3 次补仓，防止断网。
