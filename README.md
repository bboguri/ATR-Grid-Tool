## 📘 ATR 动态网格实战 SOP

### 1️⃣ 标的诊断 (判定“心跳”)
在开启网格前，必须通过 **ATR 占比 (ATRP)** 确定标的属性。
- **公式**: $$ATRP = \frac{ATR(14)}{Price} \times 100\%$$

| ATRP 数值 | 波动等级 | 风险倍数 (M) | 确认比例 (R) | 核心建议 |
| :--- | :--- | :--- | :--- | :--- |
| **< 1.5%** | 低波动 | 2.0 | 0.5% | 空间小，不建议作为主战标的 |
| **1.5% - 3.0%** | 中波动 | 1.5 | 0.8% | 稳健标的，适合宽幅箱体操作 |
| **3.0% - 5.0%** | 高波动 | 1.2 | 1.0% | **黄金标的！** 回本效率最高 |
| **> 5.0%** | 极端波动 | 1.0 | 1.5% | 风险较大，需大幅拉开间距 |

### 2️⃣ 参数设置 (条件单填报)
根据第一阶段得出的参数，在智能条件单中填写：

- **买入步长**: $-(ATRP \times M)\%$
- **买入确认 (反弹)**: $+R\%$
- **卖出步长**: $+(ATRP \times M + 0.5)\%$
- **卖出确认 (回落)**: $-R\%$

### 3️⃣ 风险管控 (生命线)
- **破位止损**: 股价有效跌破 **MA60 (60日均线)** 立即休眠。
- **财报避险**: 财报日前 3 个交易日关闭网格。
- **压力测试**: 使用计算器核对“最大回撤”，确保现金流覆盖 3 次以上补仓。
